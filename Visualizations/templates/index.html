Course Conflict Graph

The interactive graph on this page visualizes all the courses offered for a semester at Olin College. The courses appear as vertices on the graph, and are connected if two classes meet at the same times on the same days. Using a backtracking algorithm, the vertices are properly vertex colored, such that it should not be possible to attend more than one class of the same color.
